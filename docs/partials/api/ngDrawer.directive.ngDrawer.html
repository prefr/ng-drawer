ngDrawer
directive in module ngDrawer

Description

Note: Elements needs positioning, either manual or via .ng-drawer-mounting.

Usage

as element:
<ng-drawer
       ng-drawer="{string|json}">
</ng-drawer>
as attribute
<ANY ng-drawer="{string|json}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ngDrawerstringjson

If a string is provided it will be interpreted as the side where the drawer is mounted to. If it is anything different from 'left' it will be treated as 'right'.

An object passed as json should have some the follwing properties: 

	{
		drawFrom: 'left' // default is 'right'
	}
