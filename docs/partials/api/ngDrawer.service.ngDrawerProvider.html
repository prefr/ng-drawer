ngDrawerProvider
service in module ngDrawer

Description

Use this provider to set default behavior for ngDrawer directives.

Methods

  • addSnappingFunction(name, fn)

    Adds a snapping function to the drawer config, that may be used with any drawer by referencing its name.

    Parameters
    ParamTypeDetails
    namestring

    The name of the snapping function. Use this when configuring a drawer directive.

    fnfunction

    The snapping function(delta, distance) takes two parameters. delta is the amount of pixels the drawer has been pulled out during the last 15ms. distance is how far the drawer has been pulled out altogether(percentage).

  • config(custom_config)

    Extends drawer config with provided custom config object. Doesn't do anything yet.

    Parameters
    ParamTypeDetails
    custom_configobj

    plain object
